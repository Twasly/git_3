## Lists

To add unnumbered lists, you need to highlight the elements with asterisk (*). For example, this way:
* Element 1
* Element 2
* Element 3

To add numbered lists, you just need to number the elements. For example, this way:
1. Item 1
2. Item 2
3. Item 3
